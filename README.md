# Titanic Survival Prediction - AI Capstone Project

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.

## Dataset
- Titanic Dataset (.csv)
- Contains passenger details like age, gender, class, fare and survival status.

## Technologies Used
- Python
- Pandas
- NumPy
- Machine Learning
- VS Code
- Git & GitHub

## Project Steps
1. Data Loading
2. Data Cleaning
3. Data Analysis
4. Feature Selection
5. Model Building
6. Prediction

## Files
- project.py - Main Python code
- Titanic Dataset.csv - Dataset
- app.py streamlit Application
- README.md - Project documentation
