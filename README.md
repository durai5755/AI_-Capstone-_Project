# Titanic Survival Prediction

## Project Overview
This project predicts whether a Titanic passenger survived using Machine Learning.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
- Titanic Dataset

## Machine Learning Algorithm
- Logistic Regression

## Accuracy
- 80.45%

## Conclusion
The Logistic Regression model achieved an accuracy of 80.45% in predicting passenger survival.
